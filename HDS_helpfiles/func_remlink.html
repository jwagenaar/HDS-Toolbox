remlink

Removes links from object

Syntax

REMLINK(OBJ, 'propName')
REMLINK(OBJ, 'propName', indeces)

Description

REMLINK(OBJ, 'propName') removes last index of the property with 'propName'.

REMLINK(OBJ, 'propName', indeces) removes the links in 'propName' at the provided 'indeces'.
