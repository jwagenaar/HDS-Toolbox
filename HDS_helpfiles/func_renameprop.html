renameprop

Changes the name of a property

Syntax

OBJ = RENAMEPROP(OBJ, 'oldName', 'newName')

Description

OBJ = RENAMEPROP(OBJ, 'oldName', 'newName') renames the property 'oldName' to 'newName' in a single or array of objects. This method only works on properties that were added to the object using the ADDOBJ, ADDLINK or ADDPROP method.
