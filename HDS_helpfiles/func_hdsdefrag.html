hdsdefrag

Minimizes disk-space occupied by '.nc' files

Syntax

HDSDEFRAG(OBJ)

Description

HDSDEFRAG(OBJ) checks all .nc files in the data tree associated with OBJ and defragments the .nc files when previous save commands resulted in unused data in the files. This is the normal behavior for the save commands as it is programmed for speed and not for file size efficiency. The HDSDEFRAG method will optimize the saved files which will result in smaller files.
