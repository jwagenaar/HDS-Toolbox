remprop

Removes dynamic property from object

Syntax

REMPROP(OBJ, 'fieldname')

Description

REMPROP(OBJ, 'fieldname') removes property with 'fieldname' from the object if the property is a dynamic property which was added using the addprop method. Therefore, no properties that contain objects added by the ADDOBJ method can be removed using this method and no properties that are defined in the object's class definition can be removed.
