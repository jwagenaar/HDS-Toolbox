remobj

Removes a linked object from the object

Syntax

OBJ = REMOBJ(OBJ, 'classname')
OBJ = REMOBJ(OBJ, 'className', INDEX)

Description

OBJ = REMOBJ(OBJ, 'classname') removes the last entry of the property with objects of type 'classname' in the current object.

OBJ = REMOBJ(OBJ, 'className', INDEX) removes the object at the INDEX from the property with objects of type 'className' in the current object.

Example

A = Animal;
addobj(A,'Experiment','name');
remobj(A,'exp');

Technical specs

1. REMOBJ can only be called on a single object.

2. REMOBJ registers the object if the object has not been registered before.

3. REMOBJ finds name of property from which to remove and loads all objects of that property.

4. REMOBJ gets the objIDS of the objects that should be removed and find the indeces of those objects in HDSManagedData.

5. REMOBJ changes the indeces of the remaining objects in HDSManagedData.

6. REMOBJ sets the remIds in HDSManagedData.

7. REMOBJ marks all touched objects as changed to correctly save the data.

8. REMOBJ removes objIds from childIds in object, removes the objIds from HDSManagedData and deletes the objects that are removed.

9. REMOBJ removes all the child-objects of the removed object from memory.

10. REMOBJ removes all the fileIds from HDSManagedData that are no longer used, reorders the remainng fileIds and reassigns fileIDs to the objects in memory.

11. REMOBJ resorts the objIds in HDSManagedData.
